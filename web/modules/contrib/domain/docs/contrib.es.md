# Módulos contrib relacionados

Varios módulos contribuidos por la comunidad amplían el módulo Domain
con funcionalidades adicionales.

## Domain Extras

**Página del proyecto:**
[drupal.org/project/domain_extras](https://www.drupal.org/project/domain_extras)

Una colección de submódulos utilitarios que complementan el módulo
Domain:

- **Domain Alias Extras** — funcionalidades adicionales para los alias
  de dominio.
- **Domain Configuration Extras** — opciones de configuración
  adicionales.
- **Domain Early Negotiation** — resuelve el dominio activo más
  temprano en la solicitud.
- **Domain Maintenance Mode** — modo de mantenimiento por dominio.
- **Domain SSO** — inicio de sesión único entre dominios.
- **Domain SSO Admin Toolbar** — integración con la barra de
  herramientas de administración para Domain SSO.
- **Domain for Page Manager** — integración de Page Manager con los
  dominios.
- **Linkit for Domain Access** — integración de Linkit con Domain
  Access.
- **Migrate Domain** — soporte de migración de dominios.
- **Migrate Domain Access** — soporte de migración de Domain Access.

## Domain Path

**Página del proyecto:**
[drupal.org/project/domain_path](https://www.drupal.org/project/domain_path)

Permite la creación de alias de ruta separados por dominio, con
integración opcional con Pathauto.

## Domain Redirect

**Página del proyecto:**
[drupal.org/project/domain_redirect](https://www.drupal.org/project/domain_redirect)

Hace que las redirecciones sean sensibles al dominio al integrarse con
el módulo [Redirect](https://www.drupal.org/project/redirect). La misma
ruta de origen puede redirigir a distintos destinos según el dominio
activo, con las redirecciones específicas de un dominio teniendo
prioridad sobre las globales.

## Domain Theme Switch

**Página del proyecto:**
[drupal.org/project/domain_theme_switch](https://www.drupal.org/project/domain_theme_switch)

Permite a los administradores asignar un tema diferente a cada dominio.
