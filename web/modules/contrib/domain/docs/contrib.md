# Related contrib modules

Several community-contributed modules extend the Domain module with
additional functionality.

## Domain Extras

**Project page:**
[drupal.org/project/domain_extras](https://www.drupal.org/project/domain_extras)

A collection of utility submodules that complement the Domain module:

- **Domain Alias Extras** — extra features for domain aliases.
- **Domain Configuration Extras** — additional configuration options.
- **Domain Early Negotiation** — resolves the active domain earlier in the
  request.
- **Domain Maintenance Mode** — per-domain maintenance mode.
- **Domain SSO** — single sign-on across domains.
- **Domain SSO Admin Toolbar** — admin toolbar integration for Domain SSO.
- **Domain for Page Manager** — Page Manager integration with domains.
- **Linkit for Domain Access** — Linkit integration with domain access.
- **Migrate Domain** — domain migration support.
- **Migrate Domain Access** — domain access migration support.

## Domain Path

**Project page:**
[drupal.org/project/domain_path](https://www.drupal.org/project/domain_path)

Allows the creation of separate path aliases per domain, with optional
Pathauto integration.

## Domain Redirect

**Project page:**
[drupal.org/project/domain_redirect](https://www.drupal.org/project/domain_redirect)

Makes redirects domain-aware by integrating with the
[Redirect](https://www.drupal.org/project/redirect) module. The same
source path can redirect to different destinations depending on the
active domain, with domain-specific redirects taking precedence over
global ones.

## Domain Theme Switch

**Project page:**
[drupal.org/project/domain_theme_switch](https://www.drupal.org/project/domain_theme_switch)

Allows administrators to assign a different theme to each domain.
