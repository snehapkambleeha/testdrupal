# Modules contrib associés

Plusieurs modules contribués par la communauté étendent le module Domain avec
des fonctionnalités supplémentaires.

## Domain Extras

**Page du projet :**
[drupal.org/project/domain_extras](https://www.drupal.org/project/domain_extras)

Une collection de sous-modules utilitaires qui complètent le module Domain :

- **Domain Alias Extras** — fonctionnalités supplémentaires pour les alias
  de domaine.
- **Domain Configuration Extras** — options de configuration
  supplémentaires.
- **Domain Early Negotiation** — résout le domaine actif plus tôt dans la
  requête.
- **Domain Maintenance Mode** — mode maintenance par domaine.
- **Domain SSO** — authentification unique entre les domaines.
- **Domain SSO Admin Toolbar** — intégration de la barre d'outils
  d'administration pour Domain SSO.
- **Domain for Page Manager** — intégration de Page Manager avec les
  domaines.
- **Linkit for Domain Access** — intégration de Linkit avec Domain Access.
- **Migrate Domain** — support de la migration de domaines.
- **Migrate Domain Access** — support de la migration de Domain Access.

## Domain Path

**Page du projet :**
[drupal.org/project/domain_path](https://www.drupal.org/project/domain_path)

Permet la création d'alias de chemin séparés par domaine, avec une
intégration optionnelle de Pathauto.

## Domain Redirect

**Page du projet :**
[drupal.org/project/domain_redirect](https://www.drupal.org/project/domain_redirect)

Rend les redirections sensibles au domaine en s'intégrant avec le module
[Redirect](https://www.drupal.org/project/redirect). Le même chemin
source peut rediriger vers des destinations différentes selon le domaine
actif, les redirections spécifiques à un domaine ayant priorité sur les
redirections globales.

## Domain Theme Switch

**Page du projet :**
[drupal.org/project/domain_theme_switch](https://www.drupal.org/project/domain_theme_switch)

Permet aux administrateurs d'assigner un thème différent à chaque domaine.
